## Usage
You need to edit the conents in the strings as per your needs to describe who you are ? & what you are ? to display in the landing page.

```JS
    <script>
      var typed = new Typed(".typed2", {
        strings: ["Devolper.", "Designer.", "Traveller."],
        typeSpeed: 80,
        backSpeed: 80,
        loop: true,
      });
    </script>
```
